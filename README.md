# vue-mdl-datepicker

> Material Design Datepicker for Vue.js

## Usage
```vue
<template>
  <mdl-datepicker></mdl-datepicker>
</template>

<script>
import MdlDatepicker from 'vue-mdl-datepicker';

export default {
  name: 'MyComponent',
  components: { MdlDatepicker },
};
</script>
```
